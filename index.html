ทายซิ... อาจารย์คือใคร?

หน้าควบคุมของอาจารย์ (Host)

1 ตั้งค่า Firebase

A
สร้าง Firebase Project

ไปที่ console.firebase.google.com → Add project → ตั้งชื่อ เช่น prof-quiz → Create project

B
เปิด Realtime Database

ใน Firebase Console → เมนูซ้าย Realtime Database → Create database → เลือก Asia Southeast → Start in test mode → Enable

C
Get Firebase Config

⚙️ Project Settings → Your apps → กด </> Web → Register → Copy firebaseConfig object → วางในช่องด้านล่าง

2 ใส่ชื่ออาจารย์ทั้ง 10 คน

3 วางรูปคำใบ้
